# 🚀 Chaos – ChatGPT Clone

Chaos is a full-stack AI-powered chat application inspired by ChatGPT. It allows users to interact with an AI model, maintain chat history, and experience a smooth, real-time typing interface.

---

## ✨ Features

* 💬 AI-powered conversations
* 🕘 Chat history support
* ⚡ Real-time typing effect
* 🎯 Clean and responsive UI
* 🔐 Secure backend API integration
* 🌐 Full-stack architecture (Frontend + Backend)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js

### AI Integration

* OpenAI API

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/chaos.git
cd chaos
```

---

### 2. Setup Backend

```
cd backend
npm install
```

Create a `.env` file and add:

```
OPENAI_API_KEY=your_api_key_here
```

Run backend:

```
node server.js
```

---

### 3. Setup Frontend

```
cd Frontend
npm install
npm run dev
```
